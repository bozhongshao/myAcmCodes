小m点秋香

Time Limit:1000MS  Memory Limit:65536K
Total Submit:30 Accepted:10 

Description 

小m听说过，唐伯虎点秋香（秋香已不在了O(∩_∩)O~）。于是，他也想来一次。 
有N（2<=N<=8000）个秋香，每个秋香都有个编号从1...N。 
现在，开始点秋香了。所有的秋香站成一行，可是小m不知道哪个是真的秋香。于是，他向华夫人苦求，在xiaom（O(∩_∩)O~）的苦求下，华夫人给小m一张表： 
每个秋香单独一行，并且有个数字mi，mi表示在这行之前有mi个秋香的编号比这行小。 
真的秋香的编号为：所有编号之和，如果是奇数，就是编号N-1。否则，就是N-2。 
为了让小m能够抱得美人归，请你帮帮他。

Input 

有多组测试用例。 
第一行为一个单独的整数N。 
接下来是N-1行的表，由于第一个位置的秋香前面没有小的了，所以第一个不在表中给出。

Output 

单独一行，真的秋香的位置。

Sample Input 


5
1
2
1
0

Sample Output 


2

Source 

小m
