最小公倍数

Time Limit:1000MS  Memory Limit:65536K
Total Submit:105 Accepted:26 

Description 

给定两个正整数，计算这两个数的最小公倍数。

Input 

输入包含多组测试数据，每组只有一行，包括两个不大于100000的正整数.

Output 

对于每个测试用例，给出这两个数的最小公倍数，且在32位整数范围之内，每个实例输出一行。

Sample Input 


10 14


Sample Output 


70

{
__int64 a1,b1,a,b,temp;
	while(scanf("%I64d%I64d",&a,&b)+1)
	{
	a1=a;b1=b;
	if (a<b) {temp=a;a=b;b=temp;}
	while(a%b!=0)
	{temp=b;
	b=a%b;
	a=temp;
	}
	printf("%I64d\n",a1*b1/b);
	}
	return(0);
}
