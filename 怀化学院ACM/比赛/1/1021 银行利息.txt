银行利息

Time Limit:1000MS  Memory Limit:65536K
Total Submit:57 Accepted:18 

Description 

某天，小A暴富了.. 
他想投资做一笔生意，怕风险，那就只好把钱存到银行里咯 
银行的利息是这样算的：利滚利 
现在告诉你 小A的本钱M(100到1,000,000) 和他准备存的年数Y(0到400)，以及年利率R(0到20)，请你帮忙算下Y年后小A能从银行取出多少钱(在32位整数范围内)？ 


Input 

输入有多个测试用例 
每个测试用例占一行，包括三个整数R,M,Y

Output 

每个测试用例输出一行，表示小A最后能从银行取到的钱 
由于小A比较有钱，不足一元的都没要了(舍零头) 


Sample Input 


5 5000 4


Sample Output 


6077


Source 

HHTC 小C
#include<stdio.h>
int main()
{int i;
__int64 s;	
double m,y,r;
	while(scanf("%lf%lf%lf",&r,&m,&y)+1)
	{
		for(i=1;i<=y;i++)
			m=m*(1+r/100);
	s=(__int64)m;
		printf("%d\n",s);
	}
}
