小C 追 MM

Time Limit:1000MS  Memory Limit:65536K
Total Submit:63 Accepted:9 

Description 

小C最近Fa sao了，感到生活为什么就这么空虚呢... 
突然灵机一动，去追个MM吧。 
假设小C 和一PLMM 都站在一条道路上(直线)，把路看成一个坐标，小C起点为x，MM的始终站在点y。 
小C的每移动一次方式有三种 
1)向前一步 x => x+1 
2)向后一步 x => x-1 
3)向前跳跃 x => 2*x 
小C现在向你求助 他 最少能通过几次移动能追到那MM。

Input 

输入数据包括多个测试用例 
每个测试用例占一行包括两个整数x,y(0<=x,y<=100,000) 


Output 

输出小C追到MM的最小步数 
如果不能追到输出-1

Sample Input 


5 9


Sample Output 


2

Hint 

5 => 10 => 9两步
